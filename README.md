# JPMorgan Chase Quantitative Research Virtual Experience

This repository contains my solutions for the JPMorgan Chase Quantitative Research Virtual Experience Program on Forage. The project focuses on commodity pricing, financial modeling, credit risk analysis, and data-driven decision making using Python.

## Overview

The virtual experience consists of four quantitative research tasks:

### Task 1: Natural Gas Price Forecasting
- Analyzed historical natural gas price data.
- Visualized market trends and seasonal patterns.
- Built a forecasting model to estimate gas prices for future dates.
- Implemented interpolation and extrapolation techniques for price prediction.

### Task 2: Commodity Storage Contract Pricing
- Developed a pricing model for natural gas storage contracts.
- Incorporated injection and withdrawal schedules.
- Considered storage capacity, operational constraints, and storage costs.
- Calculated the net value of storage contracts based on projected cash flows.

### Task 3: Credit Risk Analysis
- Built a machine learning model to estimate Probability of Default (PD).
- Calculated Expected Loss (EL) using predicted default probabilities.
- Applied data preprocessing and model evaluation techniques.
- Utilized Python and Scikit-learn for predictive modeling.

### Task 4: FICO Score Bucketing
- Implemented quantization techniques to group borrowers into risk categories.
- Generated rating buckets based on FICO scores.
- Assigned credit ratings where lower ratings indicate better credit quality.
- Created a scalable approach suitable for future datasets.
